# Frontend - Khóa Học Online

React TypeScript frontend cho hệ thống quản lý khóa học.

## Quick Start

```bash
npm install
npm start
```

Ứng dụng chạy tại: http://localhost:3000

## Cấu trúc

- `src/components/` - Reusable components
- `src/pages/` - Page components
- `src/contexts/` - Context providers
- `src/services/` - API services
- `src/types/` - TypeScript types

## Tech Stack

React 18 • TypeScript • Material-UI • React Router • Axios

## Backend API

http://localhost:8080
